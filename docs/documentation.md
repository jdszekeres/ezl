# EZL docs
### #
comment in ezl
usage
```
PRINT "HELLO WORLD"
# prints hello world
```
### PRINT
print statment in ezl
usage 
```
PRINT "HELLO WOLRD"
```
### INPUT
input statement in ezl
usage
```
PRINT "what is your favorite number"
INPUT num
PRINT num
```
### IF
if statement in ezl
usage
```
IF 10.0 == 10.0 THEN
    PRINT "you should not be seeing this."
ENDIF
```
### WHILE
while statment in ezl
usage
```
a = 0
WHILE a < 10 REPEAT
    PRINT a
ENDWHILE
```
