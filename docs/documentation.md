# EZL docs

### PRINT
print statment in ezl
usage 
```
PRINT "HELLO WOLRD"
```
### "#"
comment in ezl
usage
```
PRINT "HELLO WORLD"
# prints hello world
```
### INPUT
input statement in ezl
usage
```
PRINT "what is your favorite number"
INPUT num
PRINT num
```
### IF
if statement in ezl
usage
```
IF 10.0 == 10.0 THEN
    PRINT "you should not be seeing this."
ENDIF
```
### LET
define a varible in ezl
usage
```
LET a = 2
PRINT a
#output: 2.00
### WHILE
while statment in ezl
usage
```
LET a = 0
WHILE a < 10 REPEAT
    PRINT a
ENDWHILE
```
